This dashboard is designed and developed as part of the Smart Irrigation IoT System. The system monitors real-time soil moisture, controls pumps automatically, and provides cloud logging + visual representation of the entire irrigation process.

About the Smart Irrigation Project

The Smart Irrigation System uses an ESP32 microcontroller to continuously monitor soil moisture levels using capacitive soil sensor modules. Based on real-time readings, the system automatically decides whether to activate the pumps in Auto Mode. It also allows the user to completely override pump control in Manual Mode using Firebase.

  • ✔ Real-time monitoring (Firebase Realtime Database)
  • ✔ Auto/Manual mode switching
  • ✔ Pump override control
  • ✔ Live moisture display on dashboard
  • ✔ Visual representation via ThingSpeak charts
  • ✔ Cloud logging of moisture and pump state
  • ✔ Fully responsive web dashboard (HTML/CSS/JS)
