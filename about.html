This dashboard is developed as part of the Smart Irrigation IoT System. It monitors real-time soil moisture, synchronizes with ThingSpeak analytics, controls the pumps automatically through ESP32, and logs activities to Firebase.

About the Smart Irrigation Project

The Smart Irrigation System integrates hardware (ESP32 + Sensors + Pumps) with cloud services (ThingSpeak + Firebase) to create a fully automated irrigation platform. It uses:

  • ✔ ESP32 microcontroller
  • ✔ Capacitive soil moisture sensors (2 zones)
  • ✔ Relay-based Pump Control
  • ✔ Firebase Realtime Database
  • ✔ ThingSpeak MATLAB Analytics Engine
  • ✔ GitHub Pages Dashboard

System Architecture

The system works in a fully synchronized loop:

  • 1. ESP32 → ThingSpeak: ESP32 sends raw moisture values and pump status to the input channel.
  • 2. ThingSpeak → MATLAB Script: A MATLAB engine processes the moisture into:
    • Zone Avg Moisture (%)
    • Depletion (Dr)
    • Irrigation Requirement (mm)
    • Water Required (liters)
  • 3. ThingSpeak Output → ESP32: ESP32 fetches analytics and updates Firebase via WiFi.
  • 4. Firebase → Dashboard: This website reads live data from Firebase and shows:
    • Moisture Levels
    • Mode: AUTO / MANUAL
    • Pump Status
    • Thresholds
    • Analytics (via ThingSpeak charts)
